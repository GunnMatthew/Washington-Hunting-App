# Washington-Hunting-App
Application to easily track the hunting regulations within Washington State.

This app is currently a work in progress.  Some data might be outdated.  Verify all regulations from WDFW prior to hunting!
https://wdfw.wa.gov/hunting/regulations


# Installation Instructions
## Building from Source

### Requirements
- Android Studio
- JDK 17 or newer
- Android SDK installed through Android Studio

### Steps

1. Clone the repository:

```bash
git clone https://github.com/GunnMatthew/Washington-Hunting-App.git
cd Washington-Hunting-App
```

2. Open the project in android Studio and run via emulator

-OR-

3. Build the app using:
```
./gradlew assembleDebug
```
4. Check app/build/outputs/apk/debug/ for app-debug.apk
5. Install .apk on an Android device and run the app.
